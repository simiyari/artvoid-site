# بسته شروع Artvoid — راهنمای نصب (برای سپهر)

## این بسته چیست
چهار سند + یک فایل حافظه که Claude Code را کاملاً خودران می‌کند:

```
CLAUDE.md          ← مغز فرمانده (Claude Code خودکار می‌خواندش)
PROGRESS.md        ← حافظه پیشرفت بین سشن‌ها
docs/ROADMAP.md    ← نقشه ۶ مرحله‌ای ساخت
docs/DESIGN.md     ← سیستم طراحی مونوکروم
docs/SHOWCASE.md   ← ۲۰ ایده دموی زنده UI/UX
```

## نصب — ۳ قدم

1. یک پوشه خالی برای پروژه بساز (مثلاً `artvoid-site`) و محتویات این بسته را با همین ساختار داخلش کپی کن.
2. مطمئن شو Python 3 و Node.js روی سیستم نصب است (برای Graphify و اسکیل‌ها).
3. Claude Code را در همان پوشه باز کن و بنویس:
   ```
   بخون و ادامه بده.
   ```

همین. بار اول، Claude Code خودش وارد «حالت بوت‌استرپ» می‌شود: گیت را راه می‌اندازد،
اسکیل طراحی ui-ux-pro-max و Graphify (نقشه پروژه برای مصرف کم توکن) را نصب می‌کند،
بعد وضعیت را گزارش می‌دهد و منتظر تأیید تو می‌ماند. از سشن دوم به بعد هم با همان
«بخون و ادامه بده.» دقیقاً از جای قبلی ادامه می‌دهد (حافظه: PROGRESS.md + graphify-out).

نکته توکن: بعد از ساخته‌شدن گراف، Claude به‌جای اسکن کل فایل‌ها از نقشه
graphify-out استفاده می‌کند — مصرف توکن روی پروژه‌های بزرگ چندین برابر کم می‌شود.

## اگر وسط کار سشن جدید باز کردی
باز هم فقط بنویس: «بخون و ادامه بده.» — حافظه پروژه در PROGRESS.md است.
