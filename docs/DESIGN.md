# DESIGN.md — سیستم طراحی Artvoid

> منبع واحد حقیقت برای همه تصمیمات بصری. هر مقدار بصری در کد باید از این فایل بیاید.
> فلسفه: **مینیمال مونوکروم** — سیاه، سفید، و طیف خاکستری. سکوت بصری، دقت، اعتماد.

## ۱. رنگ‌ها

### پالت مونوکروم (تنها پالت مجاز برای UI)
| توکن | مقدار | کاربرد |
|---|---|---|
| `ink` | #0A0A0A | متن اصلی، پس‌زمینه بخش‌های تیره |
| `ink-soft` | #262626 | متن ثانویه در پس‌زمینه روشن |
| `gray-strong` | #525252 | متن کم‌اهمیت، آیکون‌های غیرفعال |
| `gray` | #A3A3A3 | placeholder، جداکننده‌های پررنگ |
| `gray-soft` | #D4D4D4 | خطوط جداکننده، border پیش‌فرض |
| `mist` | #F5F5F5 | پس‌زمینه کارت‌ها و بخش‌های ثانویه |
| `paper` | #FAFAFA | پس‌زمینه اصلی سایت |
| `white` | #FFFFFF | سطح برجسته، کارت روی mist |

### رنگ‌های سمانتیک (فقط برای وضعیت‌های سیستمی — نه دکوراسیون)
| توکن | مقدار | کاربرد |
|---|---|---|
| `success` | #16A34A | تأیید، عملیات موفق |
| `danger` | #DC2626 | خطا، حذف، هشدار بحرانی |
| `warning` | #D97706 | هشدار، توجه |

**قانون:** رنگ سمانتیک فقط در فرم‌ها، اعلان‌ها و وضعیت‌های سیستم. هرگز در دکوراسیون، دکمه‌های عادی یا تیترها.

### حالت تیره (Dark mode)
- سایت از روز اول دو حالته: `paper ↔ ink` معکوس می‌شوند
- پیاده‌سازی با CSS variables در `@theme` (Tailwind v4) + سوییچ با data-theme، نه کلاس‌های تکراری

## ۲. تایپوگرافی

- **فونت فارسی:** ایران‌یکان X واریابل با اعداد فارسی (`public/fonts/iran-yekan`) — self-hosted، وزن‌ها از محور واریابل (معادل 400 / 500 / 700)
- **فونت لاتین:** دو نقش مجزا — **Switzer** واریابل برای تیتر و لوگوتایپ (`public/fonts/switzer`) و **Inter** واریابل برای متن و اعداد لاتین (`public/fonts/inter`) — هر دو self-hosted
- **مقیاس تایپوگرافی (بر پایه 16px):**
  | توکن | اندازه | کاربرد |
  |---|---|---|
  | `display` | 56–72px (clamp) | تیتر Hero |
  | `h1` | 40px | تیتر صفحه |
  | `h2` | 28px | تیتر بخش |
  | `h3` | 20px | تیتر کارت |
  | `body` | 16px / lh 1.8 | متن (فارسی به ارتفاع خط بیشتر نیاز دارد) |
  | `small` | 14px | توضیحات، caption |
- **قوانین:** حداکثر ۲ وزن در هر صفحه استفاده شود؛ سلسله‌مراتب با اندازه و وزن، نه رنگ

## ۳. فاصله‌گذاری و شبکه

- واحد پایه: **4px** — همه فاصله‌ها مضرب ۴
- مقیاس اصلی: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128
- فاصله عمودی بین سکشن‌ها: 96px دسکتاپ / 64px موبایل
- container اصلی: max-width 1200px، padding افقی 24px
- Grid: 12 ستونه دسکتاپ، 4 ستونه موبایل

## ۴. اجزا (Primitives)

- **گوشه‌ها:** `radius-sm: 6px` (input، دکمه) / `radius-md: 12px` (کارت) — هیچ مقدار دیگری
- **Border:** همیشه 1px با `gray-soft`؛ حالت hover → `gray-strong`
- **سایه:** بسیار محدود — فقط یک سایه ظریف برای عناصر شناور (dropdown، modal). کارت‌های عادی با border جدا می‌شوند نه سایه
- **دکمه‌ها:** دو نوع فقط — `primary` (پس‌زمینه ink، متن سفید) و `ghost` (شفاف با border). هر دو با ترنزیشن 200ms

## ۵. حرکت (Motion)

- **شخصیت حرکتی:** آرام، دقیق، لوکس — هیچ چیز نپرد و ندرخشد
- ترنزیشن پایه: `200ms ease-out` (hover) / `400–600ms cubic-bezier(0.22,1,0.36,1)` (ورود عناصر)
- الگوهای مجاز: fade + رشد ملایم (scale 0.98→1)، حرکت عمودی کوتاه (8–16px)، reveal متن خط‌به‌خط
- Scroll animations: با Intersection Observer، فقط یک بار اجرا شوند
- **الزامی:** همه انیمیشن‌ها داخل `@media (prefers-reduced-motion: no-preference)`

## ۶. زبان بصری برند

- عکاسی: مونوکروم یا با رنگ‌های بسیار خاموش؛ حس ادیتوریال
- آیکون‌ها: خطی (stroke) با ضخامت 1.5px — یک ست واحد (پیشنهاد: Lucide)
- لوگو: تایپوگرافیک [تصمیم باز]
- فضای سفید سخاوتمندانه = امضای برند

## ۷. دسترسی‌پذیری (حداقل‌های الزامی)

- کنتراست متن اصلی حداقل 7:1، متن ثانویه حداقل 4.5:1
- همه عناصر تعاملی: focus state مشخص (outline با ink)
- اندازه لمسی حداقل 44px در موبایل
- HTML سمانتیک، aria-label برای دکمه‌های آیکونی
